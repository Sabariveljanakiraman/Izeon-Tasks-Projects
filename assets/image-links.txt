VELMART - Image Links Used In This Project
=============================================

I could not download real image files into this folder, so here are
the direct links to every image used in the website. These are
placeholder images from picsum.photos (a free random image service).
They will load automatically in the img tags as long as the computer
running the website has internet access.

If you want, you can download these yourself and save them in this
"assets" folder with the same names, then just change the "img" path
in js/data.js and the "background-image" in css/style.css to point to
your local files instead (example: assets/book1.jpg).

ICONS USED
------------------------------------------------
No separate icon image files were used. All icons in this project
(cart, phone, email, location, book, delivery, price, tick mark)
are simple emoji characters typed directly in the HTML, for example:
🛒 📞 ✉️ 📍 📚 🚚 💰 ✅
This keeps the project simple and avoids needing extra icon files.
